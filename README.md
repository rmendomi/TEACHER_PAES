## 🎯 Funcionalidades Principales

### 1. Explicar Conceptos Difíciles
- Conexión con conocimiento previo
- Progresión de simple a complejo
- Ejemplos contextualizados a Chile
- Verificación de comprensión

### 2. Identificar Errores Comunes
- Reconoce patrones de error típicos
- Explica la causa del error
- Enseña cómo evitarlo
- Propone ejercicio de refuerzo

### 3. Generar Ejercicios Personalizados
- Ajustados al nivel del estudiante
- Variados y no repetitivos
- Progresión clara de dificultad
- Formato tipo PAES

### 4. Desarrollar Estrategias de Estudio
- Planes realistas según tiempo disponible
- Priorización inteligente de contenidos
- Técnicas probadas (Pomodoro, repaso espaciado, etc.)
- Manejo de ansiedad ante pruebas

### 5. Mantener Tono Pedagógico Efectivo
- Motivador sin condescendencia
- Profesional pero accesible
- Empatía con ansiedad de pruebas
- Normaliza el aprendizaje lento

## 🚀 Casos de Uso

### Estudiante Preparándose para PAES
```
"No entiendo cómo funcionan los logaritmos"
↓
Claude explica conectando con exponentes, proporciona ejemplos, verifica comprensión
```

### Estudiante con Error Conceptual Persistente
```
"¿Por qué mi respuesta a esta inecuación es incorrecta?"
↓
Claude identifica el error (no cambiar desigualdad), explica por qué, muestra correcto
```

### Estudiante que Necesita Practicar
```
"Dame 5 ejercicios de probabilidad, nivel intermedio"
↓
Claude genera 5 ejercicios variados con progresión clara
```

### Estudiante Ansioso
```
"Me pone muy nervioso entrar al examen"
↓
Claude enseña técnicas de respiración, gestión de tiempo, preparación mental
```

### Profesor Preparando Material
```
"Necesito 10 ejercicios de Lectura Crítica nivel intermedio"
↓
Claude genera ejercicios variados con clave de respuestas y notas pedagógicas
```

## 🎓 Filosofía Educativa

Esta skill se basa en:

1. **Pedagogía Constructivista**
   - Construir sobre conocimiento previo
   - Aprender haciendo
   - Significatividad

2. **Teoría de Mentalidad de Crecimiento**
   - "No entiendes *aún*"
   - Valoración del esfuerzo
   - Aprendizaje de errores

3. **Ciencia Cognitiva**
   - Repaso espaciado
   - Interleaving
   - Elaboración

4. **Empatía Pedagógica**
   - Normalizar dificultades
   - Reconocer ansiedad
   - Celebrar progresos
